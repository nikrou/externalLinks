Ce plugin permet d'ouvrir les liens externes dans une nouvelle fenêtre

Il ajoute une petite icône juste après les liens.

Par défaut, lorsqu'on clique sur le lien, le contenu de la page courante est remplacé par le contenu de la page cible. Si on clique sur l'icône on ouvre une nouvelle page.

Usage

Activer le plugin dans l'onglet paramètre.

Vous pouvez :

  • ouvrir tous les liens dans une nouvelle fenêtre : cochez la première case.
  • ou vous pouvez choisir d'ouvrir chaque lien dans une nouvelle fenêtre. Cette deuxième case à cocher définit le statut par défaut des liens ajoutés depuis l'éditeur des billets.

Pour des raisons d'ergonomie, vous pouvez choisir de n'avoir qu'un seul lien et regrouper le lien classique et celui ouvrant une nouvelle fenêtre. Ce lien ouvrira une nouvelle fenêtre.

Et pour les mêmes raisons, vous pouvez choisir de ne pas afficher l'icône.
