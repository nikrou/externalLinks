That plugin opens external links in a new window.

It adds an icon after link.

By default if you click the link, the targeted link is opened in the same page. If you click the icon, a new window is open.

Usage

Enable the plugin (tab Settings)

You can :

  • open all external links in a new window : check first checkbox
  • or choose which link to open in a new window individually. That checkbox define default status for new link added from entries' editor.

For ergonomic reason, you can merge the two links. So there will be one link and it will open a new window.

For same reason, tou can choose to not show icon for external links.
